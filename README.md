## Dependencies
OracleDBViewer requires the following libraries:

* [libcex](https://github.com/patrickjane/libcex)
* [nlohmann/json](https://github.com/nlohmann/json)

## Installing
### server

```sh
$ cd server
$ make
```
### client
```sh
$ cd client
$ npm install
```

## How to use
### server
```sh
$ cd server
$ ./main
```
### client
```sh
$ cd client
$ npm run start
```
open browser http://localhost:3000
